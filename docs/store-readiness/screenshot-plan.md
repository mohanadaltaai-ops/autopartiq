## Google Play Assets

### Required

- App icon: 512x512
- Feature graphic: 1024x500
- Phone screenshots: at least 2, recommended 6-8

### Suggested Google Play Screenshot Size

Use phone screenshots in portrait layout.

Recommended:
- 1080x1920
or
- 1242x2688

## Apple App Store Assets

### Required

- App icon: 1024x1024
- iPhone screenshots for required device sizes

### Suggested Apple Screenshot Size

Use high-quality portrait screenshots.

Recommended:
- 1290x2796 for 6.7-inch iPhone display
or
- 1242x2688 if using generated/app preview frames

## Screenshot Production Notes

- Use light mode for primary screenshots.
- Use clean demo data.
- Avoid real customer/supplier private data.
- Use market-specific names:
  - Auto Parts IQ for Iraq
  - Auto Parts AE for UAE
- For AE screenshots, use AED currency and UAE wording.
- For IQ screenshots, use IQD currency and Iraq wording.
- Keep visual language consistent with the approved mockup.
- Avoid showing temporary Vercel URLs in screenshots.
- Avoid showing development/debug UI.
